Three steps

01 · install

git clone github.com/Yau-427/imprint then copy skill/imprint into ~/.claude/skills/

02 · generate

Run /imprint in Claude Code. Everything is computed locally; your data never leaves your machine.

03 · publish

Open a PR adding your page at claude-code-profiles/p/<your-github-login>/index.html — it appears here after merge.
