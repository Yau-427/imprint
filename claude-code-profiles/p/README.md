# Publish your profile

1. Generate your card with the [imprint skill](../../skill/imprint/) (`/imprint` in Claude Code).
2. Open a PR adding **one directory named exactly your GitHub login**:

```
claude-code-profiles/p/<your-github-login>/index.html
```

Your page must be self-contained (single HTML file; web fonts allowed) and produced by the skill's privacy pass: own projects by name only, client/NDA work anonymized, no secrets, no third-party personal data. PRs that violate this are declined.

After merge it appears at `iris.direct/claude-code-profiles/p/<your-github-login>/` and in the gallery.
